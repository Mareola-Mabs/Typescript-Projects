# Task Manager App (TypeScript)

A simple and lightweight **Task Manager application** built with **TypeScript** and **Vite**.  
The app allows users to manage daily tasks with persistent storage using **Local Storage**.

---

## Features

- Add new tasks
- Mark tasks as complete
- Delete tasks
- Tasks persist using **Local Storage**
- Fast development setup with **Vite**
- Written in **TypeScript** for type safety

---

## Tech Stack

- **TypeScript**
- **Vite**
- **HTML / CSS**
- **Local Storage API**

---

Tasks are stored in the browser using Local Storage, so they persist even after refreshing the page.

No backend or database is required.
